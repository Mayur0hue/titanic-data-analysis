\# Titanic Data Analysis



\## 📌 Project Overview

This project also includes a basic machine learning model to predict passenger survival.

This project analyzes the Titanic dataset to understand the factors affecting passenger survival.



\## 📊 Dataset

\- Source: Kaggle Titanic Competition

\- Link: https://www.kaggle.com/competitions/titanic

\- Files used: train.csv



\## 🛠 Tools \& Technologies

\- Python

\- Pandas

\- Matplotlib

\- Seaborn

\- Jupyter Notebook



\## 🔍 Steps Performed

\- Data Cleaning (handling missing values)

\- Exploratory Data Analysis (EDA)

\- Feature Engineering

\- Model Building



\## 📈 Key Insights

\- Female passengers had higher survival rates

\- Passengers in higher class had better survival chances

\- Age and fare influenced survival probability



\## 🤖 Model Performance

\- Public Score: 0.7799
